# Signal Store Lab

A hands-on Angular 20.3.17 application for learning NgRx Signal Store from scratch.

## Prerequisites

- Node.js 20.19+ or 22.12+
- npm 10+

## Run

```bash
npm install
npm start
```

Open `http://localhost:4200`.

The app demonstrates `withState`, `patchState`, `withComputed`, `withMethods`, and root-level store scope. The task board is powered by `src/app/task.store.ts`.
